# Console Game Suite — Semester 1

Three classic games built from scratch in C++ with no graphics libraries.

## Games
- Snakes & Ladders — dice rolls, player movement, multiplayer support
- Nokia Snake — ASCII graphics, grow mechanic, collision detection
- Gomoku (Five in a Row) — two-player, win condition checks, customizable board
